INTRODUCTION
------------

The Drupal date popup authored module provides a jQuery UI datepicker for the 
"Authored on" date field found on node forms.

For a full description of the module, visit the project page:
  http://drupal.org/project/date_popup_authored

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/date_popup_authored

REQUIREMENTS
------------

- Drupal 6.x
- date_popup, part of the date module (http://drupal.org/project/date); 6.x-2.x branch.

INSTALLATION
------------

Install as usual, see http://drupal.org/node/70151 for further information.
